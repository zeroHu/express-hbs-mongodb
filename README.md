### express+mongoose 请求数据库简易框架
------
#### 主要内容
* request 请求别的服务器的资源
* router 分离 restful 接口
* 模块化views页面
* 链接mongodb数据库，简单的存取值
* 模块化增加registerHelper

> [项目成果展示地址](http://www.zeroyh.cn)
s
```
准备工作
    安装mongodb
项目运行
    git clone https://github.com/zeroHu/express-hbs-mongodb.git
    cd express-hbs-mongodb
    npm install
    npm start
```
